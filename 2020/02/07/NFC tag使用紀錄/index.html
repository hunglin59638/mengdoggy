NFC tag使用紀錄

Posted on 2020-02-07 Edited on 2021-03-26 In technology

最近發現原來NFC感應也可以讓使用者設定要執行的指令

首先要先準備nfc tag作為手機感應的媒介

但上面這部影片使用的app只有ios有而已，所以裡面提到的快捷指令和toggl兩個app都無法使用
ios版的toogl設定中有個功能可以設定在siri中說了哪個關鍵字後開啟計時某個項目
然後再用快捷指令這個app去設定siri啟動計時的指令輸入nfc tag中
最後再用手機去感應nfc tag就可以計時了

Time tracking
於是我也來找可以用android手機可以做到time tracking的app
Timesheet - Time Tracker:
功能除了基本記錄某個項目的時間還有時薪之類的，可用在計算自己薪水哈
Timesheet NFC Add-On:
這個才是主要的app，同個團隊開發的，可以將要啟動的項目作成NFC tag

大概都是些自己摸一摸就會的設定就不多說了

另一個也很有趣的是NFC tools pro
網路上有它的付費破解版嘿嘿…
NFC Tools PRO 6.1.1

目前想到的應用有

  1. 自動連上某個Wifi，順便輸入密碼
    Add a task > Networks > Wi-Fi network
  2. 製作自己的名片，別人一掃就知道電話跟e-mail
    這個最簡單的方式就是在設定連結讓別人到你的blog或FB就好，但也只是玩味興趣偏重
    或是把文字輸入到剪貼簿，別人按貼上就可以。使用時機還是很少的
