選擇筆記軟體

Posted on 2019-12-09 Edited on 2021-03-26 In technology

Introducion（自己的廢話就放到最後）

Joplin

  • 同步可以使用多種雲端服務，Onedrive、dropbox等都可以
  • 有Web clipper，需先開啟port才能網頁擷取，而且擷取的很標準
  • 多平台支援，三大作業系統也有手機版
  • 筆記本可以開子分頁，該筆記本本身也可以新增頁面
  • 頁面可以選擇要作為note或任務，也可以之後轉換，任務其實就只是標題前有個check box而已，其他跟note是一樣可以編輯內容
    為目前首選的筆記軟體了，可以滿足大部分的需求，有支援網頁版就更棒了

Laverna

  • 免安裝，有Web版
  • 缺點是同步的雲端目前只有dropbox和私有雲端
  • Dropbox API自己操作無法使用成功，即使成功也頂多只有6GB。如果後來又建立NAS就會是不錯選項了
    從onenote過來，所以會習慣在軟體裡直接插入圖片，但後來發現markdown跟html的格式可以插入網頁圖片並直接瀏覽
    所以把圖片上傳到google再插入連結就不會佔用太多容量了

Vnote

  • 介面尚有些凌亂，欄位很多但有些華而不實
  • 只支援dropbox同步
  • 免安裝

安裝指令

版本若有更新要至新的網址修改一下下載網址

https://github.com/tamlok/vnote/releases

將下載的執行檔改為可執行並執行

如何在 Ubuntu 18.04 上安装 VNote

Leanote

  • 安裝過程看起來就很複雜了，於是就先放棄他了

私有雲筆記服務：Leanote（螞蟻筆記）
自建雲筆記–螞蟻筆記LEANOTE

Boostnote

  • 介面簡潔，還可以客製化，不過沒有功能欄，基本上就是Markdown語法套入，所以要很熟悉不然很痛苦…
  • Windows免安裝
  • 同步方式是掛在一個雲端硬碟，然後把儲存筆記的資料夾設在裡面，不過這樣跟本地備份就很像，如果今天在第三方電腦還是不太方便
  • 筆記型式有Markdown或程式碼兩種，程式碼可以精準套色給不同語言的code，中可以看
  • 我目前的第2首選，可惜在於記事本只能有一層，不能創建子層
  • 有看到Web版，但應該還在開發中，有雲端版可以登入，但需要用github登入，登入時會出現錯誤的回報
  • 近期更新版本的樣子，應該是在推雲端編輯，只是是由他們提供雲端空間100MB/月免費，要再多就要加錢(看來中國開發的筆記軟體都這樣…)

QOwnNotes

  • 目前用起來還不錯，發現開源的筆記軟體分記事本的方式是直接用主機中目錄區分
  • 不像onenote包成一個筆記本檔案。程式在windows以安裝啟動，只是一個執行檔方便以usb攜帶。
  • 同步平台支援Nextcould或Owncloud，也就是有NAS了…
    Install QOwnNotes in terminal

    https://www.qownnotes.org/installation

Notable

一款精美的基於Markdown的筆記應用程式——Notable

  • 支援Linux、MacOS和Windows
  • 使用上很多不直觀，要在功能欄的View切換成預覽編輯結果
  • 左邊的功能列隱藏也是要在View裡面看
  • 介紹文中有看到左方欄位可以建立Notebook，但我找不到使用方法

前言

現在大多數使用的筆記軟體應該就非Onenote和Evernote莫屬了

之前我也是Onenote的愛用者，因為可雲端同步，容量還有30GB可以說是綽綽有餘

因為Evernote免費用戶每月上傳容量有限制，這也是選擇Onenote的一大原因

不過現在Ubuntu跟Win10都會使用，而onenote在ubuntu並沒有支援

每次要做筆記只能開啟Web版編輯，自己網速不夠下點開新的頁面都要讀取一段時間

而且現在筆記內容多有command或code，想要支援markdown文件格式

可以highlight這些內容

總之需求要有

  1. 可雲端同步
  2. 支援Win10跟Ubuntu，最好手機版跟網頁版也有
  3. code跟command可以有highlight，也就是以markdown格式編輯文件
  4. 要收錢的話也很願意，但要是一次買斷的（真的不喜歡那種訂閱制的…）

    訂閱制是哪來的邪魔歪道
