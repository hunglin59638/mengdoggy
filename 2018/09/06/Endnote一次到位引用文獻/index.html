Endnote一次到位引用文獻

Posted on 2018-09-06 Edited on 2021-03-26 In technology

Endnote目前更新到X9版本了

也是走最近的簡約風格，變美很多

如果不想每次都要手打一遍文獻這個軟體就會是個很好的助手

最近有種感慨，不是軟體不好，而是人不會使用而已

I. 下載Endnote

獲得Endnote的方法通常是從學校線上資源下載

附上中興大學連結，但需要連上學校wifi或使用虛擬伺服器

II. 從學術網站上下載引用檔

通常選擇RIS或Endnote file

這次以google scholar作為示範

點選 引用的圖標 ’ ’ ，點選Endnote或是RefMan，下載引用檔

III. 開啟新的reference library

全部下載完所需的引用文獻後，打開Endnote

按 file> New… 開啟一個新的reference library

IV. 匯入引用檔

雙擊檔案就即可匯入

或按Inport匯入，記得檔案類型要選對

V. 選擇引用格式

雖然軟體已經列出期刊的引用格式，但可能還是會不符合你的需求

可以再進一步編輯

Edit > Output styles > Open style manager

選擇自己所需的期刊格式，這裡以 J Animal Scienece

勾選後按右下角的 Edit

VI. 編輯期刊格式

進入後看到下方的介面，在論文後引用的文獻是在 Bibliography 編輯

進入 Author Lists，確認 Abrreviated Author List選項是 List All author names

如果沒有，另外一個選擇預設在超過6位作者就會用 et al.簡寫

Author Name，第一位作者開頭是姓再來是名，其他作者先列名再列姓

Capitalization選擇 Normal，表示每個字第一個字大寫，之後小寫

Initials縮寫格式為 A. B.

Title Capitalization 選擇 sentence style capitalization

表示文獻標題只有第一格單字第一個字母會大寫，其餘單字小寫

關閉儲存修改結果

VII. 匯入期刊縮寫

如果這時直接匯出會發現引用的期刊名稱顯示全名沒有縮寫

Endnote有內建多種領域的期刊縮寫清單

按 Tools > Open Terms Lists > Journals Term list

先把子項目 Terms 中的期刊名(你匯入的期刊) 用 Delete Term 篩除

換到子項目 List ，選擇 Import List…

沒有移動安裝位置的話應該會在 C:\Program Files (x86)\EndNote X9\Terms Lists

可以看到有很多txt文字檔

選擇自己期刊領域的文字檔，這裡選擇Zoological Records

回到Term可以看到相關領域期刊的全名跟縮寫都已匯入

VIII. 匯出引用文獻

在左上方下拉式選擇修改後的期刊格式

按Import

選擇匯出存檔類型為 Rich Text Format (*rtf)

參考網站

臺大圖書館參考服務部落格

EndNote匯出參考文獻至word(改格式)
